# HTML(Hyper Text Markup Language)

--- 

- 마킹을 해주면 컴퓨터가 읽고 무슨 역할을 하는 지 알 수 있다 

## 태그

- head: 문서에 관한 정보, 화면 밖의 모든 것

- body: 사용자가 눈으로 보는 모든 것, 화면 안에 보이는 모든 것

- div: 영역을 구분해주는 것?

- 주의) 태그 선택자로 css style 만들지 말기 예) div{ width: 100;}



## 블록, 인라인, 인라인 블록

-  블록: 넓이, 높이 지정 가능
-  인라인: 널이 높이 지정 불가능
